Secondly, define where to put all modules in _moduleLayout variable, e.g.,

...
{id:'m101',col:'c1',tab:'t1'},
{id:'m102',col:'c2',tab:'t1'},
{id:'m103',col:'c3',tab:'t1'},
...
  • id - the module ID
  • col - short for column, defines which column this module goes to, c1, c2 or c3
  • tab - which tab the module belongs to
